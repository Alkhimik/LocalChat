To start the client:

-Use your terminal window

-Type the following: java -jar BroChat.jar --server-port=8080

-To have it working you need to use your server port. Otherwise it will ask you to change the port.

-After that you need to type 'logIn'(or 'in') - if you are already a user
 or 'logUp'(or 'up') - if you want to create an account

-To exit type : Exit


-To fully check functionality create more terminal windows and init more clients at once.
