To start the  server:

-run Postgresql to create db for server
-use script.sql in postgres database(this script is also represented in source code(schema.sql))


-Use your terminal window
-Type the following: java -jar ChatServer.jar --port=8080
-Port number can be chosen yourself
-If you see 'Success!' then it works
-One server is all you need.

-To close Server type Exit
